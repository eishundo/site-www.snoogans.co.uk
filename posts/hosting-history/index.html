Hosting History

    Introduction:

    I have hosted snoogans.co.uk for many years.

    jay.snoogans.co.uk

    I built this server using old server parts whilst working at the ISP, Netscalibur.

    I was lucky enough to have full access to the datacentre and hosted this for free.

    I hosted the webserver via the Apache HTTP Server (“httpd”) and a full mail server setup running:

    • Postfix.
    • Cyrus IMAP.
    • Amavisd-new, ClavAV and SpamAssassin.

    s1.snoogans.co.uk

    When I left Netscalibur/Claranet, I had to decom jay.snoogans.co.uk, so I purchased a new 1U aluminium rack mount server and hosted this with Exonetric. This hosting company is run by a few ex-colleagues from Netscalibur.

    I still hosted my own web and mail servers.

    s2.snoogans.co.uk

    To reduce monthly costs, I switched from a physically hosted server (colo) to running snoogans.co.uk inside a FreeBSD container/jail. Again, hosted with Exonetric.

    I believe at this time I moved to GMail to host my email rather than run my own mail server.

    s4.snoogans.co.uk to s5.snoogans.co.uk

    After many years running my own server and software, I decided to make life easier for myself and move to a fully hosted platform. I picked GoDaddy and let them worry about OS/software updates.

    s6.snoogans.co.uk

    Present day. Where is it hosted?
